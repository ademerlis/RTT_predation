This repository contains data and analysis scripts for the manuscript:

## Coral predation across multiple gradients: Implications for survivorship and reef restoration
#### **Authors:** Allyson DeMerlis, Adam McAnally, Amanda Bourque, Martine D’Alessandro, David S. Gilliam, Erinn M. Muller, Kristene T. Parsons, William C. Sharp, Joshua D. Voss, Diego Lirman
#### **Target Journal:** _PNAS_

-----

### Description:
This repository contains the code used to quantify corallivorous fish predation on nursery-raised corals (*Montastraea cavernosa*, *Orbicella faveolata*, *Pseudodiploria clivosa*) outplanted across 24 reef sites in six regions of the Florida Reef Tract, and to identify the drivers of predation prevalence using generalized linear mixed models, random forest, and PCA.

### Contents:

#### Data (`input_files/`):
* **Supplementary Table 1 - Coral Outplant and Fish Data.xlsx:** Tidied coral outplant monitoring and fish survey data (sheets: `tidyCoral RT_Outplant_Corrected`, `tidy Fish RT_DETAIL`, `Site Coordinates`, `cleaned_genotype_data`).
* **RT_DRM_Colonies_UM.xlsx:** DRM coral colony-level survey data.
* **RT_DRM_General_UM.xlsx:** DRM general survey data (rugosity, depth, habitat, coral counts).
* **DRM_site_rugosity_depth.csv:** Site-level rugosity and depth summary derived from the DRM surveys.
* **Region Site Metadata:** Site-level metadata (region, site, depth, strata, coordinates) referenced across analysis scripts.
* **tidy_outplant_monitoring_days_survival_glm.csv:** Tidied outplant monitoring data with survival and predation variables, as used in the response GLMMs.

#### Analysis (numbered to reflect pipeline order):
* **1_site_map.Rmd:** Generates the map of outplant site locations across the six regions.
* **2_predation_analysis.Rmd:** GLMMs of predation prevalence and survivorship; builds the Predation Index by region.
* **3_fish_analysis.Rmd:** GLMMs of parrotfish and butterflyfish counts across regions and strata; post-hoc pairwise comparisons.
* **4_RF_PCA_analysis.Rmd:** Random forest variable importance and PCA of predation drivers, incorporating DRM coral cover/density data.

#### Figures (`plots/`):
* Figures generated by the numbered analysis scripts above.

#### Output Tables (`results/`):
* Summary and statistical output tables (GLMM ANOVA, fixed effects, emmeans contrasts, PCA loadings/variance explained, Predation Index by region), generated by scripts 2–4.
